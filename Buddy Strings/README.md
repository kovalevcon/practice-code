# Buddy Strings

Source: [Link](https://leetcode.com/problems/buddy-strings/)

## Task:

Given two strings `A` and `B` of lowercase letters, return true if and only if we can swap two letters in `A`
so that the result equals `B`.

**Sample Input**

```
A = "ab", B = "ba"
```

**Sample Output**
```
true
```
